# Media boundary

Real-house footage can make the demo more convincing, but raw and review media never
belongs in Git. Keep capture files under the ignored `media/raw/`, `media/review/`, or
`media/exports/` directories and publish only the final, privacy-reviewed video through
the submission platform or an approved video host.

The repository may later contain synthetic diagrams, product screenshots, and generated
visual assets that have passed the same privacy review. See
[`docs/REAL-FOOTAGE-CHECKLIST.md`](../docs/REAL-FOOTAGE-CHECKLIST.md).
